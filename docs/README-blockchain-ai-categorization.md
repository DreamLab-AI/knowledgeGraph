## 🔍 Categorization Methodology

### Bitcoin-Specific Criteria
A file is classified as Bitcoin-specific if it meets ANY of:
1. **High Bitcoin keyword count** (≥3 occurrences)
   - Keywords: bitcoin, btc, lightning, rgb, l402, x402, satoshi, taproot, etc.
2. **Lightning Network protocols** (L402, X402)
3. **RGB Protocol** (Bitcoin smart contracts)
4. **Bitcoin mining context** (SHA-256 + mining, difficulty, mining pools)

### General Blockchain Criteria
A file is classified as general blockchain if it meets ANY of:
1. **Ethereum-specific** (≥2 Ethereum keywords)
   - Keywords: ethereum, eth, erc-20, erc-721, solidity, gasper, etc.
2. **Other blockchain platforms**
   - Polkadot, Cosmos, Algorand, Hyperledger, etc.
3. **Generic blockchain concepts** (no Bitcoin specificity)
   - Generic smart contracts, multi-chain infrastructure
